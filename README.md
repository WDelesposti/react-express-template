# Quick Start

### 1. Prerequisites
- Node.js (v20.17.0)
- npm

### 2. Back-end Setup
```bash
cd backend
npm install
npm start
```

### 3. Front-end Setup
```bash
cd frontend
npm install
npm start
```

### 4. Use the app
- Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
